# Binary Operators

binary operators as defined in section 12.3 of ISO 10303-21

## Overview

In addition to the relational operators, defined in 12.2.1.2, two further operations are defined for binary data types: indexing ([ ]) and concatenation (+) .

## Topics

### Binary Indexing
as defined in section 12.3.1 of ISO 10303-21

- ``SDAI/BinaryType/subscript(_:)-(I?)``
- ``SDAI/BinaryType/subscript(_:)-(Int?)``
- ``SDAI/BinaryType/subscript(_:)-(ClosedRange<Int>?)``

- ``...(_:_:)-(T1?,_)``
- ``...(_:_:)-(T2?,_)``
- ``...(_:_:)-(_,U3?)``

### Binary Concatenation Operator 
as defined in section 12.3.2 of ISO 10303-21

- ``+(_:_:)-(TBi?,UBi?)``
